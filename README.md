# 海狸同步 HailySync

## What is HailySync

HailySync is a cloud sync solution for Obsidian, designed for users who need a simple and reliable sync experience, especially in China.

## Why HailySync

- Official Obsidian Sync may have network issues in some regions
- Existing solutions are complex to set up
- Users need a stable, easy-to-use sync service

## Features

- Simple setup (no complex configuration)
- Multi-device sync
- Device binding system
- End-to-end encryption (E2EE)
- Optimized for stability and reliability

## Quick Start

1. Install the plugin
2. Open settings
3. Copy your device binding code
4. Enter the code on another device
5. Start syncing

## Security

- Your data is not exposed in plaintext
- Encryption is based on your device binding code
- You can control and reset device access at any time

## Privacy

- No user content is collected for analysis
- Only necessary data for sync is processed
- Users retain full control over their data
